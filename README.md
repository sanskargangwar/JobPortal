# Job Portal

A full-stack MERN Job Portal that connects **Job Seekers** and **Recruiters** through a modern web application. The platform provides secure authentication, job posting & management, resume uploads, application tracking, and AI-powered job recommendations.

---

## 🚀 Features

### 👨‍💼 Job Seeker
- User Registration & Login
- JWT Authentication
- Update Profile
- Upload Resume
- Browse Available Jobs
- Search & Filter Jobs
- View Job Details
- Apply for Jobs
- Track Applied Jobs
- AI-Based Job Recommendations

### 🏢 Recruiter
- Recruiter Registration & Login
- Create Company Profile
- Post New Jobs
- Edit & Delete Jobs
- View Applicants
- Accept/Reject Applications
- Company Dashboard

### 🔐 Authentication
- JWT Authentication
- Password Hashing using bcrypt
- Protected Routes
- Cookie-Based Authentication
- Role-Based Authorization
